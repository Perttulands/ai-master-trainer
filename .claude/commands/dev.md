# /dev - Start Development Server

Start the Vite development server for local development.

## Steps

1. Check if any process is already running on port 5173
2. Run `pnpm dev` to start the Vite server
3. Report the URL (http://localhost:5173)

## Notes

- The dev server supports hot module replacement (HMR)
- TypeScript errors will show in the terminal
- Port 5173 is forwarded from the container
